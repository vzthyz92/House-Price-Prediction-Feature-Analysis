# House Price Prediction & Feature Analysis

## Objective
Build a regression model to predict house prices and identify the key factors influencing property value.

## Business Problem
Accurate property valuation is essential for:
- Real estate pricing
- Investment decisions
- Market analysis

This project aims to predict house prices and uncover the most influential features affecting value.

## Dataset
Ames Housing dataset containing detailed property features (size, location, quality, etc.)

## Approach

### Data Preparation
- Handled missing values
- Encoded categorical variables
- Scaled numerical features

### Modeling
- Linear Regression
- Model evaluation using:
  - Mean Squared Error (MSE)
  - R² score

### Feature Analysis
Examined feature importance to understand price drivers.

## Key Insights

- Overall quality is the strongest predictor of price  
- Larger living areas significantly increase property value  
- Location and neighbourhood have a major impact  
- Newer properties tend to have higher prices  

## Business Applications

- Assist real estate agents in pricing properties  
- Support investment decision-making  
- Identify undervalued properties in the market  

## Tools Used
Python, pandas, scikit-learn, matplotlib, seaborn

## Results
Model successfully predicted house prices with strong performance and identified key value drivers.
